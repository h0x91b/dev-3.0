tmux icon animations

Every hand-drawn tmux glyph from the dev-3.0 terminal toolbar, layout menu, and mobile pane carousel — animated. Hover any card (or tap it) to watch the icon act out the operation it triggers. The green New Window icon is the proposed addition to the desktop toolbar.

Pane toolbar

The buttons above every task terminal — TaskTmuxControls.tsx.

Split Horizontal

Cuts the current pane in two, stacked top and bottom. The new pane opens below.

Dashed cut-line saws across, the newborn bottom half flashes green, the plus pops in.

Toolbar⌃B −

Split Vertical

Cuts the current pane in two, side by side. The new pane opens to the right.

Same saw-cut, falling top-to-bottom; the right half is born and lights up.

Toolbar⌃B |

Cycle Layout

Steps through tmux's built-in layout rotation for the current window.

The two panes physically trade places while the green arrows orbit around them.

Toolbar⌃B ␣

Zoom Pane

Expands the focused pane to fill the whole window; run again to restore the split. Also used by the un-zoom badge.

The pane inflates toward the corner brackets as the arrows fire outward.

Toolbar⌃B z

Close Pane

Kills the focused pane. Closing the last one tears down the whole session (asks first).

The X is slashed on, the pane shudders, then blinks out of existence.

DangerToolbar⌃B x

tmux Hints

Opens the cheat-sheet popover: every prefix shortcut plus the iTerm2 keyboard-mode toggle.

A sonar ring pings outward — "help radiates from here".

Layout menu

The five named presets in the tmux layout ▾ dropdown.

Tiled

Every pane gets an equal cell in a grid — the fair-share layout.

The cross draws itself, then all four cells light up in tmux order.

Layout menu⌃B M-5

Even Horizontal

Panes become full-width rows of equal height, stacked top to bottom.

Both dividers sweep in, then the rows check in one by one, top to bottom.

Layout menu⌃B M-1

Even Vertical

Panes become full-height columns of equal width, left to right.

Dividers drop in, then the columns march across, left to right.

Layout menu⌃B M-2

Main Horizontal

One big main pane on top; everything else shares a strip along the bottom.

The floor is drawn, the bottom splits, then the main pane claims the spotlight.

Layout menu⌃B M-3

Main Vertical

One big main pane on the left; the rest stack in a column on the right.

The wall goes up, the right side splits, and the main pane lights up.

Layout menu⌃B M-4
Windows & mobile

Window-level actions and the phone pane carousel — MobilePaneCarousel.tsx.

New Window New

Opens a fresh tmux window (tab) in the session. Exists on mobile — proposed for the desktop toolbar.

A whole new window pops into existence: frame springs in, tab bar draws, plus spins on.
