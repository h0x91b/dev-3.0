# UX Glossary

## Destination

A stable place in the product that users can navigate to.

## Action

A command that changes state, creates something, exports something, starts a flow, or performs work.

## Surface

A region or UI container that owns a class of interaction, such as global navigation, page toolbar, row actions, settings, inspector, or command palette.

## Primary action

The main safe action for the current screen or flow. Usually one visible primary action per screen.

## Bulk action

An action that applies to selected items. It belongs in a selection toolbar or equivalent selection-specific surface.

## Configuration

A durable change to product behavior. It belongs in settings or object settings.

## Complexity budget

A project-specific limit on visible controls, nav depth, tabs, or competing actions for a given surface.
