Added an inline git diff mode that replaces the task terminal with an in-app diff viewer. Show Diff now just opens the viewer, while Branch, Uncommitted, and Unpushed switching happens inside the diff toolbar with loading states; the changed-files popup is back on the top summary badge and its diff action now opens the inline viewer focused on the selected file; file headers stay sticky while scrolling with tighter top attachment; and the file navigator is now a real left sidebar tree with collapsible folders, larger collapse controls, a wider column, and persistent Read state so unchanged diffs keep their reviewed markers. The top toolbar is also compressed to reclaim vertical space, while the branch diff summary stays beside the task status and the extra external mode buttons are gone.
