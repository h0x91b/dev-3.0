Added an inline git diff mode that replaces the task terminal with an in-app diff viewer for branch, uncommitted, and unpushed changes. The viewer now defaults to split mode, opens all files initially, lets each file collapse independently with a larger visible toggle, adds a GitHub-style Read checkbox that collapses and strikes through reviewed files, reuses parsed git hunks instead of recomputing file diffs in the renderer, and keeps ESC/back plus upstream/binary/large-file fallbacks without the whitespace-symbol toggle.
