Added an inline git diff mode that replaces the task terminal with an in-app diff viewer. Show Diff now just opens the viewer, while Branch, Uncommitted, and Unpushed switching happens inside the diff toolbar with loading states; the changed-files popup is back on the top summary badge and its diff action now opens the inline viewer focused on the selected file; file headers stay sticky while scrolling with tighter top attachment; and the file navigator is now a real left sidebar tree with collapsible folders, a wider column, tighter indentation, larger tree icons, persistent Read state so unchanged diffs keep their reviewed markers, bulk sidebar actions to collapse everything or mark the whole diff read/unread at once, and smarter file-jump alignment so the first click lands under the sticky toolbar instead of in random bullshit territory. Each diff file header now also shows its own +/− line stats, the top summary chips were folded into the main toolbar row with colored +/− counts, the changed-files popup uses a tighter right gutter so its action icons do not float in ugly dead space, the folder collapse affordance no longer looks like a stray mini button, the sidebar bulk-action header no longer has a washed-out translucent backdrop or oversized buttons, the diff viewer follows the app light/dark theme correctly, the old external diff tool setting plus its dead RPC paths have been removed entirely, and Global Settings now controls whether Show Diff starts in side-by-side or Unified mode.
