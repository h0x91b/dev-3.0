Added an inline git diff mode that replaces the task terminal with an in-app diff viewer for branch and unpushed changes. The viewer now defaults to split mode, opens all files initially, keeps each file header sticky while scrolling, adds a GitHub-style Read checkbox that collapses and strikes through reviewed files, reuses parsed git hunks instead of recomputing file diffs in the renderer, keeps Show Diff available even while branch status is still loading, moves the branch diff summary beside the task status, and drops the separate uncommitted diff button.
