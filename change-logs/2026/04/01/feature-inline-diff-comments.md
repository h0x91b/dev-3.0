The diff sidebar now shows compact clickable review cards with truncated comment previews, inline edit/delete actions, and a full-width XML copy button at the bottom, so reviewers can manage comments in place and paste the full review back into the agent chat; comments remain local to the current viewer session for now, and the internal data model is already range-ready so multi-line targets can be added later without reshaping stored comment state.
