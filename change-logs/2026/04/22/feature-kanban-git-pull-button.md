Added a Git Pull button to the global header, right after Project Terminal (visible whenever a project is open, not only on the Kanban board). The button runs `git pull --ff-only origin <branch>` in the project's main worktree and is enabled only when that worktree is on `main` or `master`; otherwise it is disabled with a tooltip explaining why. The result or any pull error is shown as an alert so users no longer have to open the Project Terminal for a routine pull.
