Add configurable coding agents support. The app now natively supports Claude, Codex, and Gemini as built-in agents that receive the task title as initial prompt. Users can also create custom shell agents via Global Settings, with access to task metadata through environment variables ($DEV3_TASK_TITLE, $DEV3_TASK_ID, etc.). Project Settings now uses an agent selector dropdown instead of a free-form command input, with backward compatibility for existing projects.
