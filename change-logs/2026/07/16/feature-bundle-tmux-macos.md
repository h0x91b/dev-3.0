macOS releases now bundle a self-contained pinned tmux 3.6a (statically linked libevent_core and utf8proc, only OS libraries otherwise) inside both the app bundle and the CLI tarball, so DMG and in-app-updater installs work without Homebrew or Command Line Tools. The app prefers the bundled binary over the Homebrew keg and PATH while keeping custom overrides and live-server compatibility, dev3 doctor identifies it, third-party license notices ship next to the binary, and the release pipeline gates on otool/codesign checks plus a sanitized-PATH new-session smoke test.
