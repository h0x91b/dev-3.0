Clicking a native macOS notification now reliably focuses the task that triggered it, whenever the click happens. A small compiled UNUserNotificationCenter shim gives the app real notification-click callbacks (Electrobun's API is fire-and-forget), replacing the old 3-second window-focus guess; clicks that arrive while the app has no window reopen one and still navigate to the task. Linux and permission-denied setups keep the previous focus-proxy behavior.
