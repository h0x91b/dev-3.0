Fixed and enriched the GA4 analytics. Replaced the app-stream `first_open` event with `first_visit` so the web data stream actually populates the "New users" metric, and replaced the hardcoded `engagement_time_msec` constant with real foreground time (visibility-aware) so engagement metrics are meaningful. Page views now carry rich `/app`-prefixed locations (e.g. `/app/project/<id>/kanban`, `.../task/<seq>`, `.../diff/<seq>`), keeping the app cleanly separable from the landing page in the shared property while never sending confidential project names. Task paths use the human-readable per-project sequence label (e.g. `981-1`) instead of the opaque hash, so analytics shows how deep into a project's task list people get.
