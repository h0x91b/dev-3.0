Cmd/Ctrl+K is now a "Go to project or task" palette with a mode toggle: Projects, Both, and Tasks. Tap Shift or click the footer mode switcher to cycle. Both interleaves projects and tasks in one recency timeline — Today / Yesterday / This week / Older — where a task's recency is when it was last updated and a project's is when you last viewed its board (opening a task inside a project doesn't count), so a project whose board you just opened shows up among your recent tasks. Fuzzy search throughout; task rows carry a status dot + project badge. It opens in your last-used mode (default Projects). Also reachable from Cmd+Shift+P and the View menu.
