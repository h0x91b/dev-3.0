Added VSCode-style token-DSL filters to both the Kanban board and the Active Tasks sidebar. Type or click to filter by priority:, label:, agent:, status:, is:attention and has:port — the search string is the single source of truth, so the P0–P4 priority chips, label chips and the funnel dropdown all edit the same tokens and never disagree. The board shows the most-popular labels inline with a "+N more" that opens the full list in the funnel. Facet values that contain the DSL's own quote character (labels, agents or custom columns named like He said "ship it") are now escaped so their tokens round-trip correctly instead of hiding the matching task.
