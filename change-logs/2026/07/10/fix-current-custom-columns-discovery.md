Fixed `dev3 current` so its custom-columns hint actually appears: the offline data reader used to drop the project's customColumns, so the section silently never rendered (even online). It now reads the full stored project and lists each custom column's 8-char ID + name in both online and offline modes, so agents can find the `dev3 task move --status <id>` token without reading it off the UI. Board column ordering/visibility also moved into a single shared, unit-tested helper reused by the Kanban board.
