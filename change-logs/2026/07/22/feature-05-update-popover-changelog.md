Short: Changelog preview + page redesign

The update-ready popover now shows a short "what's new" preview for the incoming version — the top new features plus a fix count — and links to the full Changelog screen. The Changelog screen itself was redesigned into a wider card-grid layout that fills the screen (entries grouped by date, laid out in responsive columns) where each entry is clickable to expand its full note (no more mid-sentence truncation). Changelog entries gained an optional short (≤6 word) title for the popover preview, embedded into the release update.json.
