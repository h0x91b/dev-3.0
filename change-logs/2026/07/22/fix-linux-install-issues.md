Short: dev3 remote on Linux

Polished the Linux + `dev3 remote` experience so dev3 runs cleanly on a fresh Linux box, fixing several issues hit while installing via Homebrew. The brew formula now ships the artifact-template next to the binary, so launching a task no longer fails with "Bundled dev3 artifact template not found" (a missing template also degrades gracefully now instead of blocking task launch). `dev3 remote` now waits for the Cloudflare tunnel to come up and prints the public URL + QR instead of a not-yet-useful local address. The AI install guide (ai-install.txt) documents `brew trust` as a required step on Homebrew 6.0+ and hands the `dev3 remote` run-step off to the user rather than having the installing agent start the server itself.
