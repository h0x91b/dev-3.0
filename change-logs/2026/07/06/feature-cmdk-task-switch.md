Cmd/Ctrl+K is now a unified "Go to project or task" palette. It leads with your active tasks across all projects, grouped by recency into Today / Yesterday / This week / Older (most-recently-updated first, with fuzzy search and a status dot + project badge per task), so Enter jumps straight to your most-recent task; projects follow below with their ⌘1–9 badges. Fast keyboard switching between tasks without leaving the keyboard. Also reachable from Cmd+Shift+P and the View menu.
