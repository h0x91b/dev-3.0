Cmd/Ctrl+K is now a "Go to project or task" palette with a mode toggle: Projects, This project (the current project's tasks), and All tasks (across every project). Tap Shift or click the footer mode switcher to cycle. The task modes group by recency — Today / Yesterday / This week / Older, most-recently-updated first, with fuzzy search and a status dot per task (plus a project badge in All tasks). It opens in your last-used mode (default Projects; This project is hidden outside a project). Also reachable from Cmd+Shift+P and the View menu.
