Made the New Task creation modal usable on narrow/mobile viewports: the fixed 520px width now clamps to the viewport, the modal scrolls instead of clipping the Save actions on short screens, and the footer action buttons wrap gracefully instead of cramming into one overflowing row. The label selector is now compact too — instead of listing every project label as an always-visible chip grid, it shows only the selected labels as chips plus an "+ Add Label" button that opens the searchable label picker (the same popover already used on task cards and in task detail).
