Added a unified cross-project Kanban to the Dashboard: a "By Project ↔ Board" view toggle flips the project-grouped list into a single board that aggregates every project's active work (To Do, Agent is Working, Has Questions, AI Review, Your Review, PR Review) into status columns, each card badged with its project. It stays live, is searchable, and clicking a card opens that task in its own project board. The choice is remembered per machine. A "New task" button on the board opens a project typeahead so you can create a task in the correct project without leaving the board.
