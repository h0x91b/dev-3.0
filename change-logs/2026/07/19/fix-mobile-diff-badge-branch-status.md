Short: Mobile diff summary badge fixed

Fixed the mobile task view never showing the diff summary badge ("N files +X −Y") in the summary bar: branch status was only fetched by the desktop git bar, which does not render on narrow viewports. The narrow TaskInfoPanel now owns a single branch-status instance that drives the badge, restores the merge-completion prompt on mobile, and pre-loads the git actions sheet so Rebase/Push/Create PR rows no longer flash "Loading branch status" on open. Also reworked the cramped diff-viewer header on phones — a slim pinned row (back, title with inline ±stats, search, files) while the mode/filter row scrolls away with the content and search opens as a full-width row — plus the task summary bar now wraps so wide diff counters stay tappable, and long file paths in the diff render as one truncating line instead of a multi-row wrap. Added a discovery tip pointing at the mobile ⋮ menu's Git section.
