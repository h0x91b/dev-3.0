The diff viewer now shows the task PR's GitHub review threads inline on their diff lines (branch mode) and a collapsible Conversation block with the PR's top-level comments — read-only, markdown-rendered, with deep links to GitHub and a show-resolved toggle. Threads that no longer match the rendered diff collapse into per-file "Outdated" groups instead of disappearing, each thread has a "Send to agent" button that types a fix prompt into the task terminal, and threads can be opted into the XML review export alongside your own inline comments with an origin marker.
