Mobile attachments: the touch key bar next to the terminal composer now has a paperclip button that opens the OS photo/file picker and drops the uploaded worktree path into the draft (or straight onto the prompt in raw mode), and pasting an image from the phone clipboard into the composer uploads it the same way. The New Task modal gained the same attach button next to the Description label in browser mode, and clipboard pastes carrying files now upload directly instead of reading the app host's clipboard.
