Refactored all per-agent launch/trust/hooks/skill differences behind a single AgentAdapter abstraction (one adapter per coding agent plus a GenericAdapter fallback), replacing the scattered `is*Command` ladders. Adding or changing a coding agent is now a one-file change and the type-checker enumerates every member to fill in. Pure adapter core lives in the shared layer; a thin backend executor applies trust/hook I/O. No user-visible change — every launch command, trust config, and hook file is byte-identical, guarded by a new golden characterization test and per-adapter contract tests.
