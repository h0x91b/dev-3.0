Fixed four UI rendering bugs found in a bug-hunt pass: dragging to reorder a task within a Kanban column dropped it at the wrong (doubled) position because the drop-index counted both the card wrapper and the card root; a drop below the visible cards in a truncated column (>15 tasks) silently moved the task into the hidden tail; the task-age badge showed "0y" for ages between 360 and 364 days; and colon-form ANSI color codes (e.g. 38:5:196) leaked as raw text in terminal previews instead of rendering as color.
