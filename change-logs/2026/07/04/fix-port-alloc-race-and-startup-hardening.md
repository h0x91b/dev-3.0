Fixed three bootstrap-layer bugs found in a bug-hunt pass: concurrent port allocation could hand two parallel task variants overlapping ports (now serialized with a cross-process file lock that re-reads assignments under the lock); the dev-channel Vite probe fetch had no timeout and could hang app startup forever if something squatted port 5173 (now bounded to 1s); and the dev3 CLI binary was copied non-atomically on every launch, so an agent exec'ing it mid-copy could hit a partial binary (now written to a temp sibling and atomically renamed into place).
