Fixed four data-layer bugs in the dev3 CLI: concurrent `note add`/`note delete`/`label create`/`label delete` no longer lose each other's writes (they now recompute inside the file lock); a too-short or ambiguous `--task`/`--project` prefix is rejected instead of silently resolving to the wrong entity; `label set` rejects unknown label IDs instead of persisting garbage; and launching a To Do task with variants now carries its notes and overview onto the variants instead of dropping them.
