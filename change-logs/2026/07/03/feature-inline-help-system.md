Added an inline help system: fast styled tooltips replace the slow native hovers on icon toolbars, small (i) icons in section headers (settings, board columns, modals, diff viewer, stats) open rich help cards explaining what each section does, and a new help mode (⇧⌘/ or Help → Explain This Screen) highlights every zone of the current screen with clickable explanations. All help content lives in one localized registry (en/ru/es), documented as UX bible §5.4.
