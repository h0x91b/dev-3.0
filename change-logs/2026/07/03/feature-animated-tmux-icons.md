Tmux toolbar icons now animate on hover, acting out the operation they trigger: splits saw their divider across, cycle-layout swaps the panes, zoom inflates the pane, close draws the X and blinks the pane out, layout presets draw their dividers, and the mobile New Window icon pops into existence. Shared glyph components live in TmuxIcons.tsx. The git action row (Show Diff, Rebase, Push, Create PR, PR + auto-merge, Merge) and the diff include-tests toggle now use matching hand-drawn animated icons (GitIcons.tsx): the diff types its added line and blinks the removed one, the rebase commit rides the rail to the tip, the push arrow flies into the cloud, the PR sparkle twinkles, the auto-merge bolt strikes, the merge commit lands, the tests flask boils. Git buttons are flex-centered and use compact labels (Diff, PR, Auto PR) so the row with icons stays no wider than before. The desktop tmux pane toolbar also gains a green New Window button (between the split buttons and the layout pill) that runs tmux new-window, with the same animated icon. The global header's Report and Changelog buttons are now icon-only (labels dropped, tooltips kept) to tighten the row. The entire top header row was then redesigned: the old mix of Nerd Font glyphs and ad-hoc inline SVGs (back, forward, home breadcrumb, project dropdown, no-sleep coffee/lock, quick shell, project terminal, pull with its refresh/success/alert states, remote QR, productivity stats, GitHub, report, changelog, overflow kebab, project-settings wrench, global-settings sliders, update-ready) is replaced by a single hand-drawn stroke set (HeaderIcons.tsx) matching the tmux and git icon style, each with a hover animation acting out its operation. A standalone design reference page lives at design/tmux-icons-animated.html.
