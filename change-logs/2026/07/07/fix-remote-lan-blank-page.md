Fixed Remote Access showing a blank page when opened directly over the LAN (non-Cloudflare) — the large JS bundle was served via Bun's zero-copy sendfile path, which drops the HTTP response headers over a real network socket on macOS, so the browser rejected it and the app never mounted. Static files are now read into memory before being sent, which keeps the response headers intact; Cloudflare tunnel access was unaffected because it reaches the server over loopback.
