Fixed ExtraKeyBar (Esc, Ctrl, Tab, Enter, arrow buttons) showing on desktop browsers in remote/web mode. The bar now uses touch capability detection (navigator.maxTouchPoints) instead of screen width, so it only appears on actual touch devices (phones, tablets) where a physical keyboard is unavailable. Also added a "Session Expired" screen for browser remote access when the QR token is invalid or expired, replacing the infinite loading spinner. Disabled aggressive caching for remote access static files to prevent stale bundles on mobile browsers. Added a 5-second timeout to getAppVersion to prevent blocking on mobile when WebSocket isn't connected. QR code in the Remote Access modal now visually fades out and shows "Connected" when a client connects, with the Copy URL button disabled — auto-refresh of QR codes stops immediately.
