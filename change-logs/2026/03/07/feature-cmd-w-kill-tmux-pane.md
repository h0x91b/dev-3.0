Cmd+W (macOS) / Ctrl+W (Linux/Windows) now kills the active tmux pane when the terminal has focus. If only one pane is open, the tmux session ends and the "session ended" UI is shown. Implemented by extending the existing `tmuxAction` RPC with a `killPane` action and intercepting the shortcut in `TerminalView` using capture-phase event handling, gated on terminal focus to avoid accidental triggers.
