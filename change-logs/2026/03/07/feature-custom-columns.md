Added custom columns to the Kanban board. Each project can define project-scoped columns with a name, color, and LLM instruction (a short hint explaining when an agent should move a task there). Custom columns appear on the board after the built-in columns and accept drag-and-drop from any other column. The `dev3 task move --status <column-id>` CLI command supports custom column IDs, and `dev3 current` shows available custom columns with their instructions. Column management UI lives in Project Settings.
