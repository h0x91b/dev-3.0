Add browser-mode RPC transport with JWT authentication and optional Cloudflare Tunnel support. The UI can run in any browser — scan the QR code from the header "Remote" button to open on your phone. Authentication uses short-lived JWT tokens (30s in QR, 30min session) with automatic refresh. Static passkey replaced with rotating tokens, replay prevention, and URL cleanup. Optional Cloudflare Tunnel integration (enable in Global Settings) provides a public https URL via cloudflared, making the app accessible from any network.
