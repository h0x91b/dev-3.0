Add browser-mode RPC transport with JWT authentication and optional Cloudflare Tunnel support. The UI can run in any browser — scan the QR code from the header "Remote" button to open on your phone. Authentication uses short-lived JWT tokens (30s in QR, 30min session) with automatic refresh. Static passkey replaced with rotating tokens, replay prevention, and URL cleanup. Optional Cloudflare Tunnel integration — toggle "Accessible from anywhere" directly in the QR modal to get a public https URL via cloudflared. If cloudflared is not installed, the modal shows installation instructions and a recheck button.
