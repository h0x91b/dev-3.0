Removed hardcoded task IDs from Claude Code hook commands. The hooks now use `dev3 task move --status X` without a task ID — the CLI auto-detects the current task from the worktree context. This eliminates noisy diffs in `.claude/settings.local.json` when switching between tasks and simplifies the hook generation code by removing the `taskId` parameter from `buildClaudeHooks`, `mergeClaudeHooks`, `writeClaudeHooks`, and `setupAgentHooks`.
