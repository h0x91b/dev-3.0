Kanban columns for todo, completed, and cancelled are now collapsible and collapsed by default, freeing horizontal space for active work columns. Hover over a collapsed column to peek inside; use the pin icon in the header to keep it open. Collapsed columns still accept drag-and-drop task moves. Expanded columns auto-distribute to fill the available screen width. Collapse state persists per project in localStorage.
