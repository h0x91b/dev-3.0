Add optional branch selector to the Create Task modal. Users can now choose an existing local or remote branch instead of always creating a new one. Includes word-boundary autocomplete filtering and a Fetch button to refresh remote branches. Multi-variant spawns create per-variant branches (e.g. feature/login-v1, -v2) from the existing branch's HEAD. User-owned branches are preserved on task completion — only dev3-created branches are deleted.
