Fix three bugs in the browser WebSocket RPC transport: (1) `trySend` retry loop now stops when a request times out or is resolved, preventing indefinite timer accumulation; (2) WebSocket no longer attempts to reconnect when the session token is absent (auth failure), stopping an infinite 401 reconnect loop; (3) `uploadImageBase64` RPC method is now declared in `AppRPCSchema` so TypeScript can type-check callers. Also fixed a non-null assertion on a potentially undefined network interface list in `getLocalIp()`.
