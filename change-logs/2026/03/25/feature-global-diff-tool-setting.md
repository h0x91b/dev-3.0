Add a global "Default Diff Tool" setting in Global Settings. Users can choose between Git terminal diff (default), VS Code, IntelliJ IDEA, WebStorm, Kaleidoscope, Beyond Compare, FileMerge, Meld, or a custom command as their diff viewer. Available tools are auto-detected on PATH; unavailable ones are greyed out. Custom commands use $LOCAL/$REMOTE placeholders for the base and changed file versions.
