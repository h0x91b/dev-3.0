Added task notes — a per-task scratchpad for markdown notes. Each note has a source (user or AI), timestamps, and auto-saves as you type. Notes are stored inline in tasks.json and accessible from the expanded TaskInfoPanel. Added CLI support for notes: `dev3 note add`, `dev3 note list`, `dev3 note delete` commands with full context auto-detection. Source defaults to "ai" in CLI for agent workflows. Also fixed terminal bottom row clipping on resize — added 16px bottom padding to body/root to compensate for ghostty-web FitAddon charHeight measurement mismatch.
