Add per-project labels for tasks. Labels have a name and color, stored in a separate JSON file per project. Tasks can have multiple labels assigned via a tag button on cards. The kanban board shows a filter bar with label badges — click to toggle OR-based filtering. Label CRUD (create, rename, recolor, delete) is managed inline from the filter bar via a popover. Labels can also be pre-selected when creating new tasks and assigned from the task info panel.
