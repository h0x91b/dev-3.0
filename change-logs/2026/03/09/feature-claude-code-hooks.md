Add automatic Claude Code hooks (PermissionRequest, Stop) to dev-3.0 worktrees for reliable task status transitions. Hooks are injected into `.claude/settings.local.json` per-worktree at launch time. The PermissionRequest hook moves the task to "Waiting for You" when the agent needs permission approval. The Stop hook moves to "Review by User" only if the task is still in "Agent is Working" status, thanks to a new `--if-status` conditional flag on the CLI `task move` command.
