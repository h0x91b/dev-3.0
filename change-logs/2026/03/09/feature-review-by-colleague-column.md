Added a new built-in "External Review" column (status: `review-by-colleague`) that sits between "Review by You" and the terminal columns. Tasks in "Review by You" are automatically promoted to "External Review" when a background poller detects an open non-draft GitHub PR for that branch. The merge polling mechanism now also monitors tasks in this new status for merge detection. A toggle in Project Settings lets teams disable this column for projects that don't use peer code review.
