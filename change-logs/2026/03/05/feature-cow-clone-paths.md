Add Copy-on-Write file/directory cloning for worktree setup. Projects can now configure clone paths (e.g. `node_modules`, `.venv`) that are cloned from the root project into each worktree using APFS `clonefile(2)` on macOS (with `cp -cR` and regular copy fallbacks) or `--reflink` on Linux. This makes worktree creation near-instant for large dependency directories. Configurable via Project Settings UI.
