Projects can now store a default diff comparison target in `.dev3/config.json` or `.dev3/config.local.json`, so branch status, rebase, and Show Diff start from the preferred remote or local base branch automatically. Project Settings now uses filterable branch pickers for the base branch and compare ref, and new repos auto-pick `main`/`master` vs `origin/main`/`origin/master` from recent git activity before saving that exact ref.
