Fix unhandled RPC timeout rejections with enriched error context. Added a Proxy wrapper around `api.request` in the renderer that catches Electrobun's generic "RPC request timed out" and re-throws with the method name (e.g. `RPC "getBranchStatus" timed out (120 000 ms)`). Added try-catch to unprotected call sites (App.tsx, TerminalView.tsx, AddProjectModal.tsx, GlobalSettings.tsx) so timeout errors are properly handled instead of becoming unhandled rejections.
