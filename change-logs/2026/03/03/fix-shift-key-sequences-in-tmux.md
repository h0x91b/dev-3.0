Fix Shift+Tab, Shift+Enter, and other Shift+functional key shortcuts not working inside tmux terminal sessions. ghostty-web's InputHandler had a shortcut path that silently swallowed the Shift modifier for functional keys (Tab, Enter, Home, End, F1-F12, etc.), sending unmodified sequences instead. Added a custom key event handler in TerminalView that intercepts Shift+functional keys and sends correct xterm-style escape sequences. Also enabled `extended-keys always` in the tmux config so CSI u sequences (like Shift+Enter) are forwarded to inner applications.
