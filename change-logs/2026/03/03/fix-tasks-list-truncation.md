Fixed `dev3 tasks list` crashing with "Invalid JSON response" when the task list is large. The root cause was Bun's `socket.write()` doing partial writes on large payloads (~8KB+), with `socket.end()` called immediately after — truncating the JSON mid-stream. Added drain-based backpressure handling on the server side, fixed UTF-8 buffer accumulation on the client side, and extracted the backpressure logic into a testable `socket-backpressure.ts` module with comprehensive unit tests.
