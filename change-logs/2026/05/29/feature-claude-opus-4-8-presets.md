Added Claude Opus 4.8 presets (model: `claude-opus-4-8[1m]`) for every existing variant — Default, Plan, Bypass, Auto, Accept Edits, Don't Ask — and made them the new defaults. The previous Opus 4.7 presets are kept as a "Opus 4.7" group at the bottom of the Claude preset list (new IDs `claude-*-opus47`) so users can still pin the older model. The preset `version` field is bumped so stored configs are reset to the new Opus 4.8 model automatically. The `--model` argument is now quoted when the value contains shell metacharacters (e.g. the square brackets in `claude-opus-4-8[1m]`), so zsh does not try to glob-expand it on launch.
