The "Create PR" button now sends a short instruction to the AI agent in the task's currently focused tmux pane, asking it to run `git push` and then `gh pr create`, instead of opening a separate git-op pane and browser. The button shows a robot icon and the label is always "Create PR" (the old "Push & Create PR" variant, its push-then-create flow, and the unused "Push & Create PR" menu item were removed). If no pane is focused, the action is a silent no-op.
