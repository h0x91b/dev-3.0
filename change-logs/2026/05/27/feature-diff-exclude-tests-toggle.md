Added an "Include tests" toggle to both the diff viewer toolbar and the task info-panel top bar. When unchecked, test files are hidden from the changed-files list and the +/− line counts are recomputed accordingly. Test detection covers a wide range of languages and frameworks: JS/TS (.test/.spec/.e2e/.cy/.bench/.stories), Python (test_*/_test/conftest/tests.py), Go, Rust, Ruby, Java/Kotlin/Scala/Groovy/C#/PHP/Swift (PascalCase Test/Tests/Spec/IT/Fact suffix), Elixir, Dart, and __tests__/__mocks__/__snapshots__/cypress/playwright/testdata/fixtures directories. The preference is persisted in localStorage and shared between both UIs.
