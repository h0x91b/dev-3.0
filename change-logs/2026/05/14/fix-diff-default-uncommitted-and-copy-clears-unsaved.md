The diff viewer now opens in Uncommitted mode by default (and remembers the last mode you picked via Branch / Uncommitted / Unpushed buttons), which fits the common "review-before-commit" workflow better than the previous Branch default. Clicking Copy to Clipboard on an inline review now clears the "unsaved review" state, so leaving the diff viewer right after copying no longer triggers the discard prompt. The prompt re-arms if you keep adding comments after the copy.
