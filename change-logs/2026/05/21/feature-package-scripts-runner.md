Added a Scripts button next to Dev Server in the task header that opens a dropdown listing all package.json scripts in the worktree. Each script launches in a new tmux pane (left/top/right/bottom split or new window), with per-task default placement and per-script overrides remembered between runs. Running scripts show a live status badge and offer focus / stop / kill actions; the runner (bun/pnpm/yarn/npm) is auto-detected from lockfiles, and multiple-lockfile collisions surface a warning. The dropdown scrolls when the package has many scripts and a backdrop closes it on any outside click. A "Run multiple scripts…" batch modal lets several scripts be launched at once, each into its own pane. Native menu adds Task → Run Script… (⌘R), Run Multiple Scripts… (⇧⌘R), Focus Running Script, and Stop Running Script entries.
