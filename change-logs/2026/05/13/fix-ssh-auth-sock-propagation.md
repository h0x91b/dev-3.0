Propagate SSH_*, GPG_*, and DISPLAY env vars into every tmux session created by the app. Without this, panes opened via split-window (git operations, dev server viewer, column agents) could inherit a stale ssh-agent socket from a long-lived tmux server, causing `git fetch`/`git push` to hang waiting for an unreachable agent. Auth vars are now explicitly set via `tmux set-environment` on task/project/home terminals and on the detached dev-server session, prepended as exports in the cleanup script, and added to tmux `update-environment` as a fallback.
