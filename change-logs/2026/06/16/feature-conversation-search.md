Added `dev3 conversations search "<query>"` to search past task conversations (completed/cancelled) by scanning each task's surviving record — the agent transcript plus the task's own notes, overview, and title/overview history — and returning the top few most relevant tasks with snippets. Ranks with BM25 (notes/overview boosted; recency tie-breaker), so common words and injected boilerplate stop dominating and a task with no surviving transcript still surfaces via its notes. Searches local files only (no app needed) across Claude, Codex, and Gemini transcripts, and hard-excludes the current task plus every sibling variant in its group so parallel variants can't peek at each other. Also exposed as a `searchConversations` RPC for a future UI panel and documented in the agent skill (on-demand use only).
