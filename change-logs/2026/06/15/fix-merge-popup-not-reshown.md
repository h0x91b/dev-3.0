Fixed the "Branch Merged — mark task completed?" popup never re-appearing when it was missed. A prompt reserved while the task was in review was muted for an hour and, if the task then left the eligible-status set (e.g. an agent resumed work, flipping it to in-progress), it was lost for good. An unanswered prompt (never dismissed) now re-offers completion as soon as the task is eligible again, instead of being time-suppressed.
