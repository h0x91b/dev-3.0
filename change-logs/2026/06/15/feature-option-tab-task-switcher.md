Added an Option+Tab task switcher: hold Option and tap Tab to flip through active tasks of the current project (Option+Shift+Tab spans all projects), release to jump. Arrows ↑↓ move both ways, Enter opens, Esc cancels. The overlay shows a live terminal snapshot, task title, and overview, ordered most-recently-used so a quick tap-tap toggles the two latest tasks. On Linux the binding is Ctrl+Tab / Ctrl+Shift+Tab since the window manager grabs Alt+Tab.
