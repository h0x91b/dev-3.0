Quick Shell (⇧⌘`) now spawns a fresh scratch task in the built-in Operations board on every press — like clicking "Scratch Task" — instead of reusing a singleton bare shell in the home directory. The new op gets the normal `Scratch — HH:mm` title, a managed work dir, and launches immediately with your default agent + config (Claude Opus 4.8 / auto by default), no agent picker. Also fixed focus on Operations ops: after the agent + split-right shell come up, focus now lands on the agent pane instead of the shell.
