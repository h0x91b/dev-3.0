Added a single source of truth for app-level keyboard shortcuts (src/mainview/keymap.ts) and a unified Keyboard Shortcuts overlay with two tabs (App + Terminal/tmux), opened via Help → Keyboard Shortcuts, the ⌘/ (Ctrl+/) shortcut, or the ⇧⌘P command palette. The previously dead Help → Keyboard Shortcuts menu item is now wired, and the standalone tmux cheat sheet was folded into the Terminal tab.
