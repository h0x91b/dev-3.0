Added a compact layout for narrow viewports (≤1600px, e.g. 14" MacBooks): the top header and the task toolbar now collapse button text labels to icon-only (tooltips preserved), and the header's external actions (Website, Report, Change Log) fold into a single "More" overflow menu so controls no longer overflow or overlap on small screens. The git action buttons (Show Diff, Rebase, Push, Create PR, PR + auto-merge, Merge) also become icon-only Nerd Font glyphs in compact mode. The redundant "Open PR" button was removed entirely — when a PR exists, the "PR #N" badge already links to it. Driven by a new useCompact() media-query hook.
