Made the Cloudflare quick tunnel the default for `dev3 remote` — the headless server now starts a public trycloudflare.com URL out of the box and includes it in the QR. Added `--no-tunnel` to opt out for local-only mode (LAN + SSH-forward only). `cloudflared` is now a Homebrew dependency of the `dev3` formula so it's always present on brew installs; tarball users get a graceful warning if it's missing. Old `--tunnel` flag and `DEV3_REMOTE_TUNNEL` env var are replaced by `--no-tunnel` / `DEV3_REMOTE_NO_TUNNEL`.
