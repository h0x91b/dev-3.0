Fixed the git pull button's loading spinner (and the identical task git-action spinners) appearing off-center. A spinning Nerd Font glyph sits in a tall advance-width × line-height box on the baseline, so animate-spin pivoted on the box center instead of the glyph's visual center and it wobbled. The spinner is now a circular ring (radially symmetric, so it rotates perfectly around its own center) rendered inside a fixed-size icon slot, so it stays centered, never shifts the icon sideways when the spin starts, and reads no larger than the neighboring toolbar icons.
