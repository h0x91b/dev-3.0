Added a Cmd/Ctrl+Shift+P command (action) palette — the action counterpart to the Cmd+K navigation palette. Fuzzy-search and run any command (theme, language, navigate back/forward, open home/project terminal, dev server, git pull, create PR, move task between safe statuses…); only commands runnable in the current context are listed. Built on a shared PaletteShell extracted from the project quick-switch palette, dispatching through the same menu router the native menu uses (which also fixed several previously no-op menu items).
