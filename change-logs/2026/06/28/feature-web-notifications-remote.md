Browser Web Notifications in remote mode: when the UI is opened in a browser via `dev3 remote`, `dev3 notify --desktop` and watched-task status/event banners now fire a real Chrome/Safari desktop notification (clickable, opens the task). Falls back to an in-app toast on insecure LAN URLs (the Notification API needs HTTPS/localhost) or when permission is denied. Added a Browser Notifications toggle in Settings → Behavior (browser-only).
