In Remote Access (browser) mode the app now renders its own File/Task/Project/View/Terminal/Help menu bar at the top of the window, since the native macOS/Linux menu bar isn't available in a browser. It is built from the same menu definition the native menu uses (now shared) and dispatches through the same action router, so behavior matches the desktop; native-only items (clipboard, window, quit) and roadmap entries are omitted, and items grey out when their context (task/project/terminal) is absent. Collapses to a single menu button on narrow/mobile screens.
