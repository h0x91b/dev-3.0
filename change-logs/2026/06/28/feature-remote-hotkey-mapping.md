Made app keyboard shortcuts transport-aware in browser remote mode: shell-only combos (⌘Q, ⌘H, New Window, zoom, hard refresh) are dropped and yielded to the browser, ⌘1–9 / ⌘N fall back to their browser-safe aliases (G then 1–9 / C), the Keyboard Shortcuts overlay shows the remote keymap with a notice, and "Open in Finder" is hidden from the command palette in the browser since it targets the server host.
