Improved the `dev3 remote` experience for headless Linux: added `dev3 remote restart` and `dev3 remote logs [--follow] [--lines <n>]`; `--static-code` is now refused on a default-on public tunnel (no replay protection) unless `--no-tunnel` is also passed; the browser "Session Expired" screen now tells you to run `dev3 remote url` on the host instead of pointing at a non-existent desktop app; the `dev3 gui` bundle URL is arch-aware (arm64 hosts no longer silently fetch the x64 bundle); and fixed stale `--tunnel`/`--detach` hints in the CLI help and console banner.
